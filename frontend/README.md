# 🎨 Frontend - Sistema de Benefícios

Aplicação Angular moderna com TailwindCSS para gerenciamento de benefícios.

## 🚀 Instalação

```bash
# Instalar dependências
npm install

# Executar em modo desenvolvimento
npm start

# Build para produção
npm run build

# Executar testes
npm test
```

## 📦 Tecnologias

- **Angular 17** - Framework principal
- **TypeScript 5.4** - Linguagem
- **TailwindCSS 3.4** - Estilização
- **RxJS 7.8** - Programação reativa

## 🎯 Funcionalidades

- ✅ Listagem de benefícios com filtros
- ✅ Criação de novos benefícios
- ✅ Edição de benefícios existentes
- ✅ Exclusão de benefícios
- ✅ Transferência de valores entre benefícios
- ✅ Busca por nome
- ✅ Filtro de benefícios ativos
- ✅ Interface responsiva e moderna

## 🌐 Acesso

Aplicação disponível em: `http://localhost:4200`

API Backend: `http://localhost:8080/api/v1`.
